# Islandora Usage Stats CSV

## Overview

Provides a CSV file containing all stats for an object collected by [Islandora Usage Stats](https://github.com/Islandora/islandora_usage_stats).

## Dependencies

* [Islandora](https://github.com/Islandora/islandora)
* [Islandora Usage Stats](https://github.com/Islandora/islandora_usage_stats)

## Installation

Install as usual, see [this](https://drupal.org/documentation/install/modules-themes/modules-7) for further information.

## Configuration

This module has no configuration options.
